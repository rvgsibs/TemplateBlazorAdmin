
# Template Blazor Admin

Template de um sistema admin usando Blazor com MudBlazor, já vem com a tela de login e uma pré autenticação.

Para efetuar login no sistema:

**Usuário:** admin

**Senha:** 123



## Funcionalidades

- Temas dark e light
- Preview em tempo real
